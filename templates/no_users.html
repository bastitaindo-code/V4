[ Akun Tidak Ditemukan ]

Login berhasil, tetapi tidak ada akun (nomor telepon) yang terkonfigurasi di dalam aplikasi.

Silakan tambahkan akun terlebih dahulu dengan menjalankan aplikasi ini dalam mode CLI (Command Line Interface) menggunakan perintah python main.py dan pilih menu untuk menambah atau mengelola akun.

Setelah akun ditambahkan, logout lalu login kembali.
